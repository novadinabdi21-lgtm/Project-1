# Soccer Quiz Game

## ideo Presentation
https://www.youtube.com/watch?v=IenIieFErUQ




## Project Overview

This project is a Python Soccer Quiz Game that asks the user 10 multiple-choice soccer questions. The program accepts the user's answers, checks whether they are correct, keeps track of their score, and displays the final results at the end. The user can choose to play again.

## Features

* 10 multiple-choice soccer questions
* User input
* Score tracking
* Displays correct answers and user guesses
* Option to play multiple times

## Python Concepts used

* Variables (Strings and Numbers)
* Lists, Tuples
* if, else statements
* for loops and while loops
* User Interaction (Accepting input and giving output)


## How to Run

1. Open the project in Python.
2. Run `Quiz_game.py`.
3. Answer each multiple-choice question by entering A, B, C, or D.
4. View your final score.
5. Choose whether to play again.
